# Saithanisch - Fachserien

## Zweck

Diese Datei bildet die zweite Ausbaustufe des Saithanischen. Auf Basis des Wurzelregisters werden hier konkrete Terminserien notiert, die direkt für Fachtexte, Glossare, Archive, Lehrhäuser und wissenschaftliche Kommentare verwendbar sind.

## Leitmuster der Serienbildung
| Muster | Funktion | Beispiel |
|---|---|---|
| `X + kaiveth` | Messung einer Grösse | `grathkaiveth` |
| `X + kaivakh` | Messinstrument für eine Grösse | `khaurkaivakh` |
| `X + aith` | Lehrgebiet, Theorie, Disziplin | `vekthaith` |
| `X + skor` / `X + skorn` | System, Feld, geordneter Raum | `thrumskor`, `mergskorn` |
| `X + ir` | Einheit, diskrete Grösse | `grathir`, `shaurir` |
| `X + veimeth` | Veränderung von X | `vekthveimeth` |
| `X + brimeth` | kritischer Schwellenwert von X | `khaurbrimeth` |
| `X + norvain` | Gesetz, Folgerelation von X | `grathnorvain` |

## Kurznotiz
- Die Serien sind offen angelegt: weitere Element-, Organ- oder Messreihen können auf denselben Mustern aufgebaut werden.
- Besonders produktiv für spätere Lehnwörter sind die Endungen `-eth`, `-aith`, `-akh`, `-ir` und die Kompositionsmuster `X + kaiveth`, `X + skorn`, `X + selthum`.
