# Saithanisch - Wissenschaftliche Beispiele

## Registerhinweise
- Die folgenden Beispiele stehen in einem streng formellen Lehr- und Kommentarton.
- Formelhafte Sprechweise bevorzugt Kopula, Gleichsetzung und kausale Ableitung.
- Symbolische Notation kann mit saithanischen Termen direkt kombiniert werden.

## Standardformeln
- `Tharneth = grath x vekthveimeth.` = Kraft = Masse mal Beschleunigung.
- `Torveth = grath / uthar.` = Dichte = Masse pro Raumvolumen.
- `Neskorveth = 1 / neskir.` = Frequenz = eins pro Zeiteinheit.
- `Khaurveimeth norvek khaurkaiveth.` = Temperaturänderung ergibt die Wärmemessung.
- `Shaurvailketh neskorveth norvek.` = Die Lichtwelle bestimmt das Taktverhältnis.

## Definitionen im Kommentarstil
- `Vaithhaurneth saithin threnaith esk.` = Ontogenese ist Teil der gelehrten Abstammungslehre.
- `Khauraith shauraithen selthin mnorum esk.` = Die Thermik ist ein mit der Optik verbundenes Fachcorpus.
- `Verthnorvain draukethis esk.` = Ein Reaktionsgesetz gilt durch Nachweis.
- `Thrumskor oarvveimethis veimath.` = Ein Organsystem soll sich an Umweltwandel anpassen.
