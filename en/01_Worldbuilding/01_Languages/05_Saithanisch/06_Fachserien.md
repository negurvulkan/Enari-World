# Saithan - specialist series

## Purpose

This file forms the second expansion stage of Saithan. Based on the root register, specific series of dates are noted here, which can be used directly for specialist texts, glossaries, archives, teaching centers and scientific commentaries.

## Guiding pattern of series formation
| Pattern | Function | Example |
|---|---|---|
| `X + kaiveth` | Measuring a size | `grathkaiveth` |
| `X + kaivakh` | Measuring instrument for a size | `khaurkaivakh` |
| `X + aith` | Teaching area, theory, discipline | `vekthaith` |
| `X + skor` / `X + skorn` | System, field, ordered space | `thrumskor`, `mergskorn` |
| `X + ir` | Unit, discrete size | `grathir`, `shaurir` |
| `X + veimeth` | Change of X | `vekthveimeth` |
| `X + brimeth` | critical threshold of X | `khaurbrimeth` |
| `X + norvain` | Law, consequence relation of X | `grathnorvain` |

## Quick note
- The series are open-ended: additional series of elements, organs or measurements can be built on the same patterns.
- The endings `-eth`, `-aith`, `-akh`, `-ir` and the composition patterns `X + kaiveth`, `X + skorn`, `X + selthum` are particularly productive for later loanwords.
