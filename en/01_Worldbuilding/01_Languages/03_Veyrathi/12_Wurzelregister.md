# Veyrathi - Root Register

## Purpose

This register collects the currently most important productive roots of the Veyrathi. It is not a full lexicon, but a guide document for future word formation. New lexemes should, if possible, be built from these roots, their derivatives and the existing composition patterns.

This version contains 190 central roots.

## Status key

- `Kern` = freely productive main root of the standard
- `System` = preferred root for formal, technical or administrative language education
- `Altform` = lexicalized older form; received, but not the first choice for new derivations
- `Basisnomen` = nominal basal root with high compositional power

## Work rules

- New words should first be checked against this register.
- If a meaning is already covered by a core or system root, no new root is created.
- If possible, new lexemes should be connected as hyponyms to existing generic terms such as `kavar`, `kovath`, `rathen`, `nimen`, `lirun` or `kevar`.
- Register synonyms are permitted if at least one form is clearly marked as core/system and the other as old form, everyday form or high register form.
- Limited homonymy is permissible if free vs. bound morphology, part of speech slot or fixed compositional structure reliably separate the reading.
- Old forms are retained for style, register or historical depth, but should not continue to grow unmarked.
- New derivations must be compatible with the sound canon from `11_Klangkanon.md`.

## Dismantling note

The next useful expansion of this register is now:
- Agriculture, horticulture and further animal care
- Medicine, nursing and emergency communication
- High register for poetics, ritual, law and regional variation

New roots should only be included in this register if they:
- are mentioned several times in the lexicon,
- have at least one clean derivation family,
- and fulfill the sound canon without special rights.
